About the Project

This project uses GraphDB and semantic technologies to represent and explore knowledge about space, including:

Celestial bodies
Space missions
Astronauts
Space agencies
Spacecraft
Galaxies

The ontology uses GraphDB for storage and retrieval, with data obtained from real APIs:

Solar System OpenData SpaceX NASA Exoplanet Archive Wikipedia
